# Agent Tabs

[English](README.md) · **Русский**

Все ваши чаты с **Claude Code** — наверху, как **вкладки браузера**. С панелью закладок, поиском и группировкой по проектам.

Читает историю сессий напрямую из `~/.claude/projects/*/*.jsonl`.

![Agent Tabs](media/screenshot.png)

## 🔒 Приватность

**Всё работает локально. Расширение не отправляет никуда ничего.** Нет ни телеметрии, ни аналитики, ни сетевых запросов — вообще. Оно только читает файлы истории Claude Code, которые уже лежат на вашем диске, и рисует их в панели. Код открыт — можно проверить: сеть в нём не используется ни строчкой.

## Что умеет

- 🗂 **Вкладки как в браузере** — открытые чаты живут вкладками наверху, переключение одним кликом.
- ★ **Закладки** — звёздочка в чате закрепляет его на панели закладок (сохраняется между запусками).
- 🔎 **Поиск и список** слева — все сессии, сгруппированы по проектам; поиск по названию, первому сообщению и пути.
- 🤖 **Автозаголовки** — берутся из `ai-title`, который генерирует сам Claude Code.
- 🛠 **Читаемая лента** — служебные tool-результаты не мусорят в чате, вызовы инструментов схлопнуты в компактные чипы.
- ↗ **Быстрые действия** — открыть папку проекта в новом окне, показать `.jsonl` в Finder.
- ⚡ **Не боится гигантов** — сессия на 850 МБ открывается за ~50 мс: стриминговое чтение с ранней остановкой.

## Установка

**Из Open VSX** (Cursor, VSCodium, Windsurf, Gitpod):
найдите «Agent Tabs» в поиске расширений.

**Из `.vsix`** (любой VS Code):
скачайте файл из [Releases](https://github.com/SquirrelX11/agent-tabs/releases) → Extensions → `···` → **Install from VSIX**.

Затем: `Cmd+Shift+P` → **Agent Tabs: открыть панель чатов**.

## Разработка

```bash
git clone https://github.com/SquirrelX11/agent-tabs.git
cd agent-tabs
code .
# F5 — откроется Extension Development Host, панель появится сама
```

Сборки нет — расширение на чистом JavaScript.

Собрать пакет: `npm run package` → `agent-tabs-0.1.0.vsix`

## Настройки

| Настройка | По умолчанию | Описание |
|---|---|---|
| `agentTabs.projectsDir` | `~/.claude/projects` | Где искать сессии. |
| `agentTabs.maxMessagesPerChat` | `800` | Лимит сообщений на чат (защита от огромных сессий). |

## Планы

- Провайдеры для **Cursor** (SQLite в `workspaceStorage`) и **Cline / Roo**.
